# Collective Social API

The backend API for the Collective Social application.

## Getting Started

### Prerequisites

- Docker
- Node.js and npm

### Installation & Running

1. Clone the repository
2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the Docker services:
    ```bash
    docker:up
    ```

4. Run the development server:
    ```bash
    npm run dev
    ```

The API should now be running and accessible.
